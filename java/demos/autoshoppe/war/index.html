Google App Engine - Technology Sample

AutoShoppe

Spring MVC, Spring ORM, Spring DAO, Spring Core, Spring AOP, JSON, JDO, JavaMail

AutoShoppe : This application demonstrates a couple of business use cases involved in an online used car store.

The Application runs on Google App Engine and uses Spring framework for navigation, rendering views, AOP, ORM and DAO support. It also demontrates how to integrate various Google App Engine services with the Spring DI container.
