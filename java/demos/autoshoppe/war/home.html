This is a demo of a used car store. The application uses spring framework to drive the backend AutoShoppeService. This example uses Spring MVC framework to render JSON views asynchronously to the client. The server side code running on Google App Engine runs within Spring Dependency Injection Container. The persistence layer uses Datastore API to persist and fetch entities. A single MultiActionController drives the navigation by mapping URLs to method names. View the dependency graph.
